## 📌 주요 기능
### 로그인
- 아이디, 비밀번호 입력 시 로그인 가능
- 로그인 시 쿠키(Cookie) 및 세션(Session) 생성

### 회원가입
- 아이디, 비밀번호, 성별, 이메일, 주소 입력 시 회원가입 가능

### 게시판
- 게시물 작성, 읽기, 수정, 삭제(CRUD)
- 게시물 페이징 처리
- ChatGPT를 활용한 챗봇과의 대화

### 게시물 상세
- 댓글 등록, 읽기, 수정, 삭제 (CRUD)
- 댓글 무한 스크롤 처리
